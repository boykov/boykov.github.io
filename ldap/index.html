OpenLDAP 2.3 и CentOS 5

FROM centos:centos5

В CentOS 5 из базового репозитория устанавливается 2.3.43 версия OpenLDAP. 

RUN yum -y update && \
    yum -y install openldap-servers openldap-clients && \
    yum clean all

Там, где это не вызывает недоразумений, вместо синтаксиса обычных команд иногда будет использоваться синтакс Dockerfile. Например, предыдущая команда означает просто yum -y install openldap-server openldap-clients .

Для запуска сервиса ldap необходимо подготовить каталог и файл конфигурации базы данных. 

    cp /etc/openldap/DB_CONFIG.example /var/lib/ldap/DB_CONFIG
    chown ldap. /var/lib/ldap/DB_CONFIG

    rm -f /etc/openldap/slapd.conf
    cp /root/slapd.conf /etc/openldap/slapd.conf
    chown ldap. /etc/openldap/slapd.conf

    service ldap start
    sleep 3

    killall slapd
    oldpath=`pwd`
    cd /etc/openldap
    mkdir slapd.d
    slaptest -f slapd.conf -F slapd.d
    chown -R ldap:ldap slapd.d
    chmod -R 0750 slapd.d
    mv slapd.conf slapd.conf.bak
    cd $oldpath
    service ldap start
    sleep 3
